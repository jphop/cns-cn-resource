# cns-cn-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容涉及通用资源与信息展示。仓库不针对任何特定网站或域名，仅作为静态页面的集合与分发点。

## 目录说明

- `pages/`：存放所有独立 HTML 页面文件
- `assets/`：页面所需的样式、脚本、图片等资源文件
- `docs/`：相关文档或说明文件（可选）

每个页面均为独立完整结构，可单独打开使用。

## 页面归档说明

- 页面以 `index_*.html` 或 `page_*.html` 格式命名
- 归档内容保持原始结构，不进行修改或重定向
- 页面中不包含分析代码、跟踪脚本或外部依赖（除非必要）
- 所有资源尽量本地化，减少外部请求
